# 成都农村产权交易所官网 - 产品需求文档

## 1. 产品概述

成都农村产权交易所官网是展示成都市农村产权交易信息的官方门户网站，旨在为广大农户、新型农业经营主体、投资人提供权威的产权交易信息服务。

**核心目标：**
- 打造权威、专业的农村产权交易信息发布平台
- 提供清晰的交易指南和政策法规查询服务
- 树立成都农村产权交易所的官方形象
- 促进农村产权交易公开、公平、公正进行

**目标用户：**
- 农户及农村居民
- 新型农业经营主体（合作社、家庭农场、农业企业等）
- 潜在投资人
- 相关政府部门和机构

---

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览公开信息、查看交易公告 |
| 注册用户 | 手机号注册 | 提交产权交易申请、查看更多详情、订阅推送 |

### 2.2 功能模块
1. **首页** - 展示交易所概况、交易数据、推荐项目、新闻动态
2. **关于我们** - 交易所简介、组织架构、资质荣誉、发展历程
3. **交易项目** - 土地经营权、林权、集体资产等产权项目展示与查询
4. **交易指南** - 交易流程、所需材料、常见问题
5. **政策法规** - 国家及地方相关政策法规文件
6. **新闻中心** - 交易所动态、行业资讯、通知公告
7. **联系我们** - 联系方式、地理位置、在线留言

---

## 3. 核心页面说明

### 3.6 新闻中心页面
- 新闻分类（交易所动态、行业资讯、通知公告）

- 联系方式卡片（电话、邮箱、地址）

## 4. 用户交互流程

### 4.1 浏览项目流程
```
访问首页 → 进入交易项目列表 → 筛选/搜索项目 → 查看项目详情 → 联系交易所
```

### 4.2 提交交易申请流程
```
注册/登录 → 选择项目 → 填写申请信息 → 提交申请 → 等待审核 → 审核结果通知
```

## 5. UI设计规范

### 5.1 设计风格
- **风格定位**：专业、权威、可信赖的政府/机构风格，融合现代简约美学
- **整体氛围**：清新、专业、接地气，体现农业特色和地域文化

### 5.2 色彩方案
| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色调 | #1B5E20 | 深绿色，代表农业、生态 |
| 辅助色 | #8D6E63 | 棕色，代表土地、自然 |
| 强调色 | #FF8F00 | 金橙色，代表丰收、希望 |
| 背景色 | #FAFAFA / #FFFFFF | 浅灰/白色，简洁大方 |
| 文字色 | #212121 / #616161 | 深灰/中灰，层次分明 |

### 5.3 字体规范
- **标题字体**：思源黑体（Noto Sans SC）或阿里巴巴普惠体
- **正文字体**：思源宋体（阅读感）或系统默认
- **字号层级**：H1 48px、H2 36px、H3 24px、正文 16px、辅助文字 14px

### 5.4 布局规范
- **桌面端**：1200px 最大宽度，响应式适配
- **导航**：固定顶部导航，滚动时添加阴影
- **内容区**：左右留白，图文比例协调
- **卡片**：圆角8px，轻微阴影，白底展示

### 5.5 交互动效
- **页面切换**：淡入淡出，300ms
- **卡片悬停**：轻微上浮 + 阴影加深
- **按钮交互**：背景色渐变，微缩放
- **数据展示**：数字滚动动画

### 5.6 图标与图片
- **图标库**：Remix Icon 或 Lucide Icons
- **图片风格**：真实场景为主，田园风光、农产品特写
- **图片处理**：圆角、轻微阴影

---

## 6. 响应式策略

- **桌面优先**（1200px+）
- **平板适配**（768px-1199px）：导航折叠，网格调整
- **移动端适配**（<768px）：汉堡菜单，单列布局，触摸优化
